# Introduction 
O jogo deverá sortear uma fruta conforme a lista abaixo:

banana
jabuticaba
pitanga
mirtilo
morango
abacaxi
cereja
O objetivo é acertar o nome da fruta. O jogador informa uma letra e o jogo verifica se a fruta tem essa letra.
